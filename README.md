# pokemons
# Название проекта
**Сервис Pokemons**
Интерактивное приложение для просмотра покемонов и добавление покемонов в список "Избранное". При клике на изображение покемона на главной странице появляется детальное описание каждого покемона.

## Технологии
 * React
 * React hooks
 * CSS Flex Layout
 * Адаптивная верстка
